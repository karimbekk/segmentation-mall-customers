🧠 **Real-Time Customer Segmentation Dashboard**
**Project Overview**

This project is an interactive Machine Learning dashboard that performs real-time customer segmentation using the K-Means clustering algorithm.
It allows dynamic exploration of customer groups and instant classification of new customers through an interactive interface.

**Project Objectives**

- Apply unsupervised learning for customer segmentation

- Build an interactive ML-powered dashboard

- Enable real-time prediction

- Provide dynamic cluster visualization

- Demonstrate end-to-end ML integration

**System Architecture**

User Input -> Data Preprocessing -> K-Means Model -> Cluster Prediction -> Visualization

**Key Features**

- Adjustable number of clusters (K)

- Interactive 2D and 3D visualizations

- Real-time classification

- Clean and responsive UI

- Modular code structure

**Tech Stack**

- Programming Language: Python
- Machine Learning: Scikit-learn
- Data Processing: Pandas, NumPy
- Visualization: Plotly
- Web Framework: Streamlit
